Overview

The Magento PWA project is a set of developer tools that allow for the development, deployment, and maintenance of a PWA storefront on top of Magento 2.

Basic Concepts

Things you need to know to work with the Magento PWA Studio tools.

PWA Buildpack

Tools and Libraries

Utilities provided by the Magento PWA project.

Peregrine UI Components

Venia Theme

The Venia theme is a proof-of-concept Magento theme built using PWA Buildpack tools and Peregrine components.

Contribute to the Project

The Magento PWA project is an open source project that welcomes community contributions and collaboration.

If you have any questions, please join the #pwa Slack channel at magentocommeng.slack.com.
